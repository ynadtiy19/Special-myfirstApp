## 💡 **How to Get Started**  
Follow these simple steps to clone, set up, and run the project locally.

### 1. Clone the Repository  
```bash
git clone https://github.com/ynadtiy19/deyu9887-myfistApp.git
```

### 2. Install Dependencies  
```bash
flutter pub get
```

### 3. Run the Project  
```bash
flutter run
```

### 4. Build for Release  
```bash
flutter build apk
```
